# 🐾 Bud AI - Emotion-Responsive Study Assistant

Bud AI is an interactive, AI-driven study companion developed for **Protothon 2026**. It features an emotion-responsive mascot designed to make learning more engaging through personalized interaction.

##  Key Features
- **Emotion-Responsive Mascot**: A primary cat mascot that reacts to user input and touch interactions.
- **AI Study Integration**: Powered by Google Gemini to assist with academic queries and environment management.
- **Premium UI/UX**: A dark-themed, glassmorphism interface designed for a modern, high-end feel.
- **Responsive Design**: Optimized for seamless use across different devices.

##  Technical Stack
- **Frontend**: Next.js, React, Tailwind CSS.
- **Animations**: Three.js, @splinetool/react-spline.
- **AI Engine**: Google Gemini API.
- **Tools**: VS Code, GitHub.

##  Getting Started

### 1. Clone the Repository
```bash
git clone [https://github.com/jouzia/Bud-AI_.git](https://github.com/jouzia/Bud-AI_.git)
cd Bud-AI_
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Environment Setup
Create a `.env.local` file in the root directory and add your API key:
```env
NEXT_PUBLIC_GEMINI_API_KEY=your_gemini_api_key_here
```

### 4. Run Locally
```bash
npm run dev
```

##  Project Structure
- `/app`: Application logic and page routing.
- `/public`: Static assets including mascot models and images.
- `/styles`: Global CSS and glassmorphism styling.
